# Laser-cutter-engraver ⚙️

This is a laser cutter project designed by myself. The idea came from the needing such tool and from not wanting to spend a load of money on prebuilt ones.

## Overview

**1. Tools**
* 3D printer,
* Hex screwdrivers,
* soldering iron

**2. Parts**
* Laser module,
* 2020 and 2040 aluminium extrusions, roller gantrys
* 3x Nema 17 stepper motor,
* Arduino Uno + CNC Shield + 3 stepper drivers,
* MGN12 rail,

For the exact parts I've used, please refer to the BOM.

**3. Software**
GRBL, Universal G-Code Sender, Arduino IDE, Fusion 360
